## Purpose

W7 Night Wolves Bot connects TikTok creator features with a Discord community server for the W7 Night Wolves gaming community.

The bot is used for community management, creator notifications, live announcements, and content organization.

The bot is not used for spam, fake engagement, automated likes, automated follows, automated comments, mass messaging, or view manipulation.
